Responda

1. ¿Cuál es la forma correcta de aplicar una Media Query en CSS para dispositivos con un ancho de pantalla mayor o igual a 768 píxeles?

a. @media (min-width: 768px)
a. @media (max-width: 768px)
a. @media (width: 768px)

2. ¿Qué propiedad de CSS se utiliza dentro de una Media Query para aplicar estilos específicos a dispositivos con una relación de aspecto determinada?

a. aspect-ratio
a. device-width
a. orientation

3. ¿Cuál es la sintaxis correcta para combinar varias Media Queries en una regla CSS?

a. @media (condition1) and (condition2) { ... }
a. @media (condition1, condition2) { ... }
a. @media (condition1) or (condition2) { ... }

4. ¿Cuál es la Media Query que se puede utilizar para aplicar estilos a dispositivos de alta resolución (como pantallas Retina)?

a. @media (min-resolution: 2dppx)
a. @media (max-resolution: 2dppx)
a. @media (resolution: 2dppx)

5. ¿Cuál es la Media Query que se utiliza para aplicar estilos específicos a dispositivos que están en modo apaisado (landscape)?

a. @media (orientation: portrait)
a. @media (orientation: landscape)
a. @media (orientation: auto)
